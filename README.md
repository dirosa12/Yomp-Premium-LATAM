# 🧪 Yomp Premium LATAM - QA Automation Project

Proyecto de automatización de pruebas para la aplicación **Yomp Premium LATAM**, enfocado en validar funcionalidades clave de negocio en aplicaciones móviles, web y APIs.

---

## 🧠 Descripción

Este proyecto contiene casos de prueba automatizados diseñados para asegurar la calidad del software mediante la validación de flujos críticos como:

- Apertura y cierre de caja  
- Registro de usuarios  
- Validación de operaciones de venta  
- Manejo de permisos del dispositivo  
- Integración con servicios externos  

El objetivo es reducir errores en producción y mejorar la estabilidad del sistema.

---

## ⚙️ Tecnologías utilizadas

- 🧪 Katalon Studio  
- 🔁 Git / GitHub  
- 🔌 Postman (API Testing)  
- 🗄️ SQL (validación de datos)
